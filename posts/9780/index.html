二级导航菜单的写法

发表于 2019-12-17 | 分类于 CSS | |
字数统计: 521 | 阅读时长 ≈ 2

二级导航菜单效果

实现思路

  1. 先写一个一级导航菜单
  2. 每个一级菜单里边嵌套一个和 <a></a> 标签同级的二级带单 ul
  3. 通过绝对定位来确定二级菜单的位置
  4. 默认情况下让 二级菜单隐藏
  5. 鼠标滑过一级菜单时显示二级菜单
