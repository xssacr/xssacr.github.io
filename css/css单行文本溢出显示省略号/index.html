overflow

overflow 溢出属性(可以设置滚动条)

属性值: scroll/auto/hidden/visiable(默认值)/inherit

  • overflow-x
  • overflow-y

white-space

空白空间的处理

属性值:

+ `normal` 默认值
+ `pre`  空白会被浏览器保留，并且不会自动换行
+ `pre-wrap` 空白会被浏览器保留，并且会换行
+ `pre-link` 
+ `nowrap` 不换行(用的最多)
